===============================================================================
                       EDUFLOW — GUÍA DE RUTA DE URLS
===============================================================================

Este documento contiene un resumen de todas las URLs importantes para el
desarrollo local del proyecto EduFlow, incluyendo rutas web, endpoints de la 
API REST, documentación interactiva y las credenciales de prueba.

-------------------------------------------------------------------------------
1. RUTAS DEL PORTAL WEB (FRONTEND)
-------------------------------------------------------------------------------
* Raíz (Redirección Inteligente al Dashboard):
  http://127.0.0.1:8000/

* Inicio de Sesión (Login):
  http://127.0.0.1:8000/accounts/login/

* Registro de Nuevos Alumnos (Signup):
  http://127.0.0.1:8000/accounts/signup/
  (Nota: Las cuentas nuevas quedan inactivas hasta la aprobación de un admin)

* Catálogo de Cursos (Mis Cursos):
  http://127.0.0.1:8000/courses/

* Detalle de Cursos (Detalle de Curso):
  http://127.0.0.1:8000/courses/<id>/

* Control de Asistencia a Sesión (Solo Admins/Profesores):
  http://127.0.0.1:8000/courses/sessions/<id>/attendance/

* Perfil del Usuario Logueado (Editar Datos):
  http://127.0.0.1:8000/profile/

* Panel de Administración de Django:
  http://127.0.0.1:8000/admin/
  (Enlace directo a usuarios: http://127.0.0.1:8000/admin/accounts/customuser/)


-------------------------------------------------------------------------------
2. ENDPOINTS DE LA API REST (DRF & JWT)
-------------------------------------------------------------------------------
* Documentación Interactiva de Swagger (Probar endpoints en navegador):
  http://127.0.0.1:8000/api/docs/

* Documentación Estructurada de ReDoc:
  http://127.0.0.1:8000/api/redoc/

* Obtención de Token JWT (Autenticación):
  http://127.0.0.1:8000/api/auth/token/ [POST]
  (Envía: {"username": "usuario", "password": "password"})

* Actualización de Token JWT (Refresh):
  http://127.0.0.1:8000/api/auth/token/refresh/ [POST]

* Endpoints del Enrutador de la API (Añadir cabecera "Authorization: Bearer <token>"):
  - Cursos:       http://127.0.0.1:8000/api/courses/
  - Sesiones:     http://127.0.0.1:8000/api/courses/<id>/sessions/
  - Auto-Matrícula: http://127.0.0.1:8000/api/courses/<id>/enroll/ [POST]
  - Asistencias:  http://127.0.0.1:8000/api/sessions/<id>/attendance/ [GET/POST]
  - Inscripciones: http://127.0.0.1:8000/api/enrollments/
  - Pagos:        http://127.0.0.1:8000/api/payments/
  - Facturas:     http://127.0.0.1:8000/api/invoices/
  - Descargar PDF: http://127.0.0.1:8000/api/invoices/<id>/pdf/
